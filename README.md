# Grandaj donajxoj (Velké dary)

Milí přátelé,
toto je webová stránka velkých darů.  
Největším darem je samozřejmě Život sám.
Za ten vděčíme tak trochu svým rodičům a dalším předkům.
Ale jak se svým životem naložíme a jaký bude, to již záleží zcela na nás.
Pokud nejsme se svým životem spokojeni, můžeme se samozřejmě vymlouvat na to,
že jsme zdědili špatnou genetickou výbavu, špatnou výchovu nebo vzdělání nebo
že jsme dostali málo peněz, ale to nám nijak nepomůže.
Dobrá zpráva zpráva je, že nic z toho pro zařízení, aby náš život byl skvělý nepotřebujeme.
A zvláště to platí, pokud ho máme ještě skoro celý před sebou.
A o tom jsou právě tyto dary jako např.:
Kaizen, Ikigai, Hara Hachi Bu, Wabi-sabi, Gambaru, Shuhari, Guoji, Ičigo zamaj, Shoshin, Do nebo Pomodoro
Tato webová stránka 
http://tiny.cc/grandaj-donajxoj
